# LabSeq
LabSeq exercise for Altice

To test the code developed, a set o commands must be executed: \n
1- On the root folder open command line and execute "mvn clean install";\n
2- After that, execute "mvn spring-boot:run";\n
3- When the application is running, the correct functioning can be tested in 2 ways:\n
    3.1- Open your preferred browser and go to "http://localhost:8080/labseq/n", replacing n with the desired number;\n
    3.2- Alternatively, open your preferred browser and go to "http://localhost:8080/index.html", this is the Web GUI that invokes the service;\n
4- To access the REST API documentation generated by Swagger go to "http://localhost:8080/swagger-ui/".
